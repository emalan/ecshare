CMS Web Application framework for creating Wicket Applications. Easy configuration via Spring files. Common application functionality enabled by using Wicket Panels.

Present configuration sets up a database (derby in development, mysql in production) and a content repository (Jackrabbit)

Functionality includes

  • Spring Configuration for JCR API to Content Management System
  • Abstract Wicket Application Classes
  • Content Panels that can be edited online
  • Dynamic Multi-lingual support for text panels
  • Site Administrative functionality including multi-lingual meta-data, admin email, supported languages
  • User administration including welcome email, password reset, use ssh or not setting
  • Configuration options for using SSH on server for login and password resets

List of Panels that can be used in web sites

  • One line "hover-click editable" text blocks
  • Multiple line text areas - editable using full html online editor includes multi-lingual translate functionality
  • Multi image albums with different configurations - fade rotate, table layout
  • Editable Link to File - hide/show functionality, currently supports pdf and doc formats
  • Blog Panel - multiple Blog functionality
  • Email Panel - email and log messages
